

# Prueba campus php 

Eesta es un prueba de campuslands para php

## Instalacion y uso

* Instala php8.
* Instalar Composer version 2.5.5
* Clona el repositorio.
* Entra a la carpeta de ru repositorio y ejecuta.
```
    composer install
```
* Usa el archivo campuslands.sql para cargar la base de datos
